# Setting up Group Policy Object (GPOs) in a Lab
## Prerequisites:
1. Windows Server Installation: Install Windows Server on a Virtual Machine or physical machine.
2. Active Directory Domain Services (AD DS): Install and configure AD DS to create a domain.
3. Group Policy Management Console (GPMC): Install GPMC on your Window Server to mangae GPOs.
4. Client Machines: join a few client machines (virtual or physical) to your domain.
